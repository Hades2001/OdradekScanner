# Odradek Scanner
[中文](./readme_zh.md) | 英文

A 3D-printed Odradek Scanner project.

![Odradek Animation](./Img/Odradek_Gif.gif)

## Introduction

**Odradek Scanner** is a 3D-printed project inspired by the iconic scanner device.  
This repository is used to organize and publish the hardware resources, controller firmware, and related documentation required to build the project.

The goal of this project is not only to recreate its appearance, but also to turn it into a truly interactive Maker project that can be assembled, operated, and further expanded.

## Features

- Odradek-style 3D-printed structure
- Open-source hardware resources
- Controller firmware source code
- Project documentation including assembly manual and BOM
- Easy to organize, expand, and customize

## Repository Structure

.
├── Hardware/   # Open-source hardware files
├── Firmware/   # Controller firmware source code
├── Mechanical/ # Mechanical structure files
└── Docs/       # Assembly manual, BOM, and related documents

## Documentation

All project-related documentation will be organized in the `Docs/` directory, including:

* Assembly instructions
* BOM
* Wiring information
* User guide
* Additional reference materials
