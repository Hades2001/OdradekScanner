# Odradek Scanner
[英文](./readme.md) | 中文

一个奥卓德克扫描仪 3D 打印项目。

![Odradek Animation](./Img/Odradek_Gif.gif)

## 项目简介

**Odradek Scanner** 是一个灵感来自经典扫描装置的 3D 打印项目。  
该仓库用于整理和发布构建本项目所需的硬件资料、控制器固件以及相关文档。

这个项目的目标不只是还原外观，而是尽量把它做成一个真正可以组装、可以运行、可以继续扩展的互动式 Maker 项目。

## 项目特点

- 奥卓德克风格 3D 打印结构
- 开源硬件资料
- 控制器固件源码
- 提供组装手册、BOM 等项目文档
- 便于整理、扩展与二次开发

## 仓库目录结构

.
├── Hardware/   # 存放所有硬件开源资料
├── Firmware/   # 存放控制器代码
├── Mechanical/ # 结构文件
└── Docs/       # 存放组装手册、BOM 和相关文档

## 文档说明

所有与项目相关的文档都会整理在 `Docs/` 目录中，包括：

* 组装说明
* BOM
* 接线信息
* 使用说明
* 其他补充资料
